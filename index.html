Conceitos básicos de HTML/CSS

HTML o "esqueleto" da nossa página web

HTML (Linguagem de Marcação de Hipertexto) é o código
que você usa para estruturar uma página web e seu conteúdo.
Por exemplo, o conteúdo pode ser estruturado em parágrafos,
em uma lista com marcadores ou usando imagens e tabelas.

CSS os "músculos" da nossa página web

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata)
é uma linguagem de estilo usada para descrever a apresentação de
documentos escritos em HTML ou XML. O CSS determina o aspeto visual
de um site, como cores, fontes, espaçamentos, tamanhos e disposição
dos elementos.
